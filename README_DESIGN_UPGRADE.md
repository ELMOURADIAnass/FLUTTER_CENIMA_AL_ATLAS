## 📊 DESIGN METRICS

### Colors
- **Primary**: Yellow #F4C518 - ~8% of design
- **Background**: Black #000000 - Main surface
- **Surfaces**: Dark gray #1A1A1A - Cards
- **Text**: White #FFFFFF - Primary content
- **Accent**: Light gray #A1A1A1 - Secondary

### Typography
- **Headings**: 700-900 weight (bold, cinematic)
- **Body**: 400 weight (readable, clean)
- **Scale**: 12-48px (proper hierarchy)
- **Spacing**: 0.3-1px letter-spacing (titles)

### Shadows
- **Small**: 8px blur, 2px offset
- **Medium**: 12px blur, 4-6px offset
- **Large**: 16px blur, 8px offset
- **Yellow Glow**: 12px blur with primary alpha

### Animation
- **Duration**: 250ms (all transitions)
- **Curve**: easeOutCubic (premium feel)
- **Transforms**: Scale, translate, color
- **Performance**: 60fps capable
